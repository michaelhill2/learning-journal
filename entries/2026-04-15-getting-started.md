## What I Learned
Starting a learning journal is a powerful way to:
- Document insights and discoveries during the learning process
- Reflect on what you've learned and how it applies to projects
- Create a searchable reference for future use
- Reinforce learning through writing and reflection

A good learning journal structure should:
- Include the date of entry
- Capture the main topic or concept
- Document key takeaways
- Include code examples when relevant
- Note any challenges encountered
- List resources used
- Plan next steps
